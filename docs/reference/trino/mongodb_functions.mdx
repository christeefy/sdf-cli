## `objectid`

Extracts the TIMESTAMP WITH TIME ZONE from a given ObjectId.

*Supported Signatures*
```sql
function objectid() returns objectid
function objectid(varchar) returns objectid
```

## `objectid_timestamp`

Extracts the TIMESTAMP WITH TIME ZONE from a given ObjectId.

*Supported Signatures*

## `timestamp_objectid`

*Supported Signatures*
