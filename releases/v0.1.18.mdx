### June 5th, 2023

We are excited to announce that this series of releases to our CLI bring some exciting fixes, updates, and improvements to SDF.

### What's New?
- Introduced additional testing coverage
- Improved Redshift support including support for reused aliases
- `sdf describe` rendering enhancements 
- Table names are now case insensitive if not surrounded by ""
- Improved error handling with code location of depencency errors
- Bug fixes and stability improvements
