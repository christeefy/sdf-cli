### May 25th, 2023

We are excited to announce that a new release of our CLI is now available, bringing some exciting updates and improvements to SDF.

### What's New?
- Introduced multiple parser errors per file 
- Added schema coercion when writing Parquet files 
- Parser now accepts dangling comma. Example (1,2,3,)
- Bug fixes and stability improvements
