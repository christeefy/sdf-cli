### September 12th, 2023

We are excited to announce that a new release of our CLI is now available, bringing some exciting updates and improvements to SDF.

### What's New?
- Implemented a small patch to skipping confirmation for deletion in deploy
- Added a fix with External Table Registration & added Local AWS CLI Test Instructions 
- General bug fixes and stability improvements
