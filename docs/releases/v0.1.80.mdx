### September 15th, 2023

We are excited to announce that a new release of our CLI is now available, bringing some exciting updates and improvements to SDF.

### What's New?
- Caching and serializing improvements
- Fixed binding of ORDER BY
- Implemented a fix with DataFusion bug
- Introduced printing out all inferred functions after successful build
- General bug fixes and stability improvements
