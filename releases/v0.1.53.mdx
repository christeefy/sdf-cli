### August 10th, 2023

We released another version of the CLI which contains many updates, bug fixes, and other improvements to SDF.

### What's New?
- Introduced `sdf show` command to show workspace configuration schemas, supported functions, and more 
- Fixed External Table Definition in YML
- Improved testing for array aggregations
- Added handling of @run_time and @run_date directives in BigQuery
- General bug fixes and stability improvements
