### August 9th, 2023

We released another version of the CLI which contains many updates, bug fixes, and other improvements to SDF.

### What's New?
- Unfied classifiers: classify, reclassify, declassify with `class.label`
- BigQuery support for relation aliased columns to re-construct a struct when the relation name is a column name 
- Added sections to all BigQuery functions
- Cleaned up CLI commands and samples
- Support for all BigQuery UDFs
- Added Named Argument Support
- Support for allowing hyphens-in-identifiers
- General bug fixes and stability improvements
