## `array_concat`


**Example/s:**
```sql examples.sql
SELECT ARRAY_CONCAT([1, 2], [3, 4], [5, 6]) as count_to_six; -- value '[1,2,3,4,5,6]'
```
#### Supported Signatures
```sql
function array_concat(array<$1>, ...) returns array<$1>
```
