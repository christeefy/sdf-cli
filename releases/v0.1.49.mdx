### August 3rd, 2023

We are excited to announce another release to our CLI which brings some exciting fixes, updates, and improvements to SDF.

### What's New?
- Improved BigQuery Table Name Parsing
- Added support for BigQuery wildcard to match all tables with the same prefix
- Fixed Arithmetic Date Operators for BigQuery
- Parallelized SQL file ingestion for 20%+ performance improvement
- General bug fixes and stability improvements
