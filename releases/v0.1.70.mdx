### August 30th, 2023

We are excited to announce that a new release of our CLI is now available, bringing some exciting updates and improvements to SDF.

### What's New?
- Restored PySpark support 
- Implemented a fix for classifier propagation
- Added an external table provider & cleaned up table registration
- Fixed an unnest issue with special handling in BigQuery
- Updated build configuration to reduce warnings 
- General bug fixes and stability improvements
