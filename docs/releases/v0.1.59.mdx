### August 16th, 2023

We released another version of the CLI which contains many updates, bug fixes, and other improvements to SDF.

### What's New? 
- Fixed issue with BigQuery function declarations 
- Fixed join using aliases to remove the duplicate column error
- Added Function overloads to Date/Datetime/Timestamp
- BigQuery string can now implicitly cast to date
- General bug fixes and stability improvements
