### May 17th, 2023

We are excited to announce that a new release of our CLI is now available, bringing some exciting updates and improvements to SDF.

### What's New?
- SDF now handles overlapping INSERT INTO and CREATE TABLE schema declarations
- Broadened test coverage of SDF codebase 
- Improved logging 
- Expanded support for AWS S3 buckets 
- Bug fixes and stability improvements
