##  date_part
<Tabs>
<Tab title="Overview">
```sql
function date_part<$1> (date varchar, part $1) returns double
	Where $1 in (date, time, datetime, timestamp)
```

Returns date truncated to unit. Valid parts are   * `year`   * `quarter`   *
`month`   * `week`   * `day`   * `hour`   * `second`   * `millisecond`   *
`microsecond`   * `nanosecond`

Example: 
```sql example.sql
select date_part('second', to_timestamp('2023-02-08T15:33:42+00:00')) as value; -- value '42' 
```

##  date_trunc
<Tabs>
<Tab title="Overview">
```sql
function date_trunc(varchar, date) returns date
```

`month`   * `week`   * `day`   * `hour`   * `second`   * `millisecond`   *
`microsecond`   * `nanosecond`

Example: 
```sql example.sql
select date_trunc('day', to_timestamp('2023-02-08T15:33:42+00:00')) as value; -- value '2023-02-08 00:00:00'
```

##  from_unixtime
<Tabs>
<Tab title="Overview">
```sql
function from_unixtime(double) returns timestamp(millisecond)
```

Returns the UNIX timestamp unixtime as a timestamp with time zone. unixtime is
the number of seconds since 1970-01-01 00:00:00 UTC.

Example: 
```sql example.sql
select from_unixtime(946713601) as value; -- value '2000-01-01 08:00:01' 
```

##  to_timestamp
<Tabs>
<Tab title="Overview">
```sql
function to_timestamp(varchar, varchar) returns timestamp(millisecond)
```

Parses string into a timestamp.   Valid Timestamp Strings:   * `YYYY-MM-
DDThh:mm:ss.nnnZ` - RCF3339   * `YYYY-MM-DDThh:mm:ss.nnn-00:00` - RCF3339   *
`YYYY-MM-DD hh:mm:ss.nnn-00:00`   * `YYYY-MM-DDThh:mm:ss.nnn`    * `YYYY-MM-DD
hh:mm:ss.nnn`   * `YYYY-MM-DD hh:mm:ss`

Example: 
```sql example.sql
select to_timestamp('1992-06-28 08:43:01') as value; -- value '1992-06-28 08:43:01' 
```
