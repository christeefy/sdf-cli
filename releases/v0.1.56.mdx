### August 15th, 2023

We released another version of the CLI which contains many updates, bug fixes, and other improvements to SDF.

### What's New?
- Split the Presto binder and removed other dialect logic in the Presto binder
- Updated build options: `--save-assembly` and `--no-cache` to unify between `sdf build` and `sdf describe`
- Added namespaced and qualified functions for BigQuery
- Parsed `create function` for BigQuery
- Fixed function format signature for BigQuery and added `current_date(time_zone)`
- Introduced unnest simple array for BigQuery
- Updated the BigQuery sample table
- Introduced support for interval expressions in BigQuery
- General bug fixes and stability improvements
