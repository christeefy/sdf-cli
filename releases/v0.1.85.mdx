### September 24th, 2023

We are excited to announce that a new release of our CLI is now available, bringing some exciting updates and improvements to SDF.

### What's New?
- Implemented runnable functions: `builtin`. New section in Function blocks: `implemented-by: builtin`
- Integrated latest SparkSQL developments and tests, including support for CTEs, EXPLODE, and lots of functions
- Spark SQL Support for TPCH and IMDB + a synthetic test
- Additional Spark cleanup 
- split grammar for `type_` in preparation for substrait types
- Enabled parallel running in depth analysis
- Added support of assembly references
- General bug fixes and stability improvements
