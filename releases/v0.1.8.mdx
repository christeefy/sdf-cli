### May 4th, 2023

<p>We released another version of the CLI which contains many updates, bug fixes, and other improvements to SDF.</p>

### What's New?
- Added additional Redshift dialect support
- Improved logging efficiency 
- General support of function overloading for special identifiers
- Added output after successful installation of SDF CLI
- Bug fixes, performance, and stability improvements
