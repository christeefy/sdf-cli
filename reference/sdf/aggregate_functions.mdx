##  approx_distinct
<Tabs>
<Tab title="Overview">
```sql
function approx_distinct(boolean) returns bigint
```

Returns the approximate number of distinct input values. This function provides
an approximation of count(distinct x). Zero is returned if all input values are
null.

Example: 
```sql example.sql
create table tbl as values (1), (5), (9); 
 select approx_distinct(tbl.column1) as value from tbl; -- value '3' 
```

##  approx_median
<Tabs>
<Tab title="Overview">
```sql
function approx_median<S> (values S) returns double
	Where S in (bigint, int, double)
```

Returns the approximate median of input values. it is an alias of
`approx_percentile_cont(x, 0.5)`

Example: 
```sql example.sql

 select approx_median(tbl.column1) as value from tbl; -- value '5'
```

##  approx_percentile_cont
<Tabs>
<Tab title="Overview">
```sql
function approx_percentile_cont<S> (values S, p double) returns double
	Where S in (bigint, int, double, float)
```

Returns the approximate percentile of input values, where percentile 'p' is
between 0 and 1 (inclusive)

Example: 
```sql example.sql
create table tbl as values (1), (5), (9); 
 select approx_percentile_cont(tbl.column1, 0.5) as value from tbl; -- value '5' 
```

##  approx_percentile_cont_with_weight
<Tabs>
<Tab title="Overview">
```sql
function approx_percentile_cont_with_weight<S> (values S, p double, weight float) returns double
	Where S in (bigint, int, double)
```

returns the approximate percentile of input values with weight, where the weight
column and percentile 'p' is a double between 0 and 1 (inclusive)

Example: 
```sql example.sql
create table tbl as values (1, 0.5), (5, 0.5), (9, 0.5); 
 select approx_percentile_cont_with_weight(tbl.column1, tbl.column2, 0.5) as value from tbl; -- value '7' 
```
