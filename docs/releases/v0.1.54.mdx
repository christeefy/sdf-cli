### August 11th, 2023

We released another version of the CLI which contains many updates, bug fixes, and other improvements to SDF.

### What's New?
- Updated BigQuery function definitions and improved markdown generation with support for cross links
- Added Error Collection in SDF CLI
- Added support for implicit unnest in depth analysis
- Fixed no cache handling
- Introduced a check to warn users of building in unsupported dialects
- General bug fixes and stability improvements
