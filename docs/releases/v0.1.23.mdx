### June 8th, 2023

We are excited to announce that this series of releases to our CLI bring some exciting fixes, updates, and improvements to SDF.

### What's New?
- Fixed non-aggregate value referenced error when selecting a normal column. Now, the column inherits the qualifier of the field in the parent's schema
- Added a way to run describe tests without a file parameter
- Fixed incorrect scoping in lineage and propagation
- Bug fixes and stability improvements
