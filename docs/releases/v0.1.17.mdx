### June 2nd, 2023

We are excited to announce that this series of releases to our CLI bring some exciting fixes, updates, and improvements to SDF.

### What's New?
- We now accumulate multiple errors when parsing functions (soon to be multi-error output)
- Added source file, line number, and column number to error outputs
- Improved support for Redshift
- Bug fixes and stability improvements
