### June 8th, 2023

We are excited to announce that this series of releases to our CLI bring some exciting fixes, updates, and improvements to SDF.

### What's New?
- Implemented proper column qualifier semantics for Redshift 
- Support for option `sdf describe --output-file-contents-on-error`
- Added a generic decimal binder which supports decimal(p,s) as a unique pattern of generic type
- Added support for Generic Integer matching and evaluating
- Upgraded and added integers to the constraint map to support integer matching 
- Support for parsing list of types separated by comma
- Bug fixes and stability improvements
