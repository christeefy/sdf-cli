### May 11th, 2023

<p>We released another version of the CLI which contains many updates, bug fixes, and other improvements to SDF.</p>

### What's New?
- Updated to [DataFusion 23](https://github.com/apache/arrow-datafusion/releases/tag/23.0.0)
- Added Cargo format requirement and reformatted Rust code
- Added build tests for TPCH
- Updated SDF CLI installation output 
- Support for Redshift ~ operator, double precision, and additional Redshift syntax support
- Added TPCDS describe tests  
- Lineage now handles two part name `(<schema>.<table>)`
- Bug fixes, performance, and stability improvements
