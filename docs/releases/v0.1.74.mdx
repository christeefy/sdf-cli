### September 6th, 2023

We are excited to announce that a new release of our CLI is now available, bringing some exciting updates and improvements to SDF.

### What's New?
- Added prioritization to error messages
- General refactoring of binding context 
- Updated and improved support for Iceberg and Iceberg Spark 
- Implemented fixes for general BigQuery support 
- Added BigQuery binary bitwise operators 
- Made BigQuery backquoting case-insensitive 
- Added support for variable as a function parameter. Ex:
<div>
`SELECT CONCAT(@codec, '-------------------')`
</div>
- General bug fixes and stability improvements
