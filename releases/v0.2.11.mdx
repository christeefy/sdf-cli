### April 19, 2024

We are excited to announce that a new minor release of our CLI is now available, bringing
 improvements, new features, and bug fixes to SDF. Interested in our product roadmap or want to 
discuss features? Reach out to your SDF point of contact to provide any points of feedback!

### Major SDF Cloud Updates - Beta

Although SDF Cloud remains in beta, major UI and feature updates were released this week.

- New folder tree structure introduced for navigating the various assets of your catalog within lineage. 
- New high level stats added to give general details about each table in the new top header bar, showing 
the fully qualified table name of the selected table at all times.
- Improved metadata panel displaying the schema and code-level metadata about each table.
- Upstream / downstream depth filter introduced to shows the number of upstream/downstream nodes from the currently selected node.


### General bug fixes and stability improvements:
- Added warnings for `no models found`
- Updates to panic handling for credentials 
- Support/Fixes for timestampadd, datadiff
- Fixes push (and --delete) error for workspaces
