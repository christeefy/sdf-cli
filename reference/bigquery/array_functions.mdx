##  array_concat
<Tabs>
<Tab title="Overview">
```sql
function array_concat(array<T>, ...) returns array\<T>
```



Example: 
```sql example.sql
SELECT ARRAY_CONCAT([1, 2], [3, 4], [5, 6]) as count_to_six; -- value '[1,2,3,4,5,6]' 
```
