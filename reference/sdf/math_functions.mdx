##  abs
<Tabs>
<Tab title="Overview">
```sql
function abs(decimal(p, s)) returns decimal(p, s)
```

Returns the absolute value of x. Input must be a number

Example: 
```sql example.sql
select abs(-1) as value; -- value '1.0' 
```
